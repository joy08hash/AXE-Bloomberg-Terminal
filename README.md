# AXE Capital Bloomberg Terminal 📊

A Bloomberg Terminal-style Excel dashboard analyzing 800 hedge fund trades 
across 5 portfolio managers from AXE Capital (FY2022–2023).

## 🔍 Project Overview
Built to simulate a real-world hedge fund analytics environment using 
advanced Excel features — pivot tables, slicers, VLOOKUP, and 
interactive charts with Bloomberg-inspired dark theme formatting.

## 📁 Sheets Included
- **Trade Ledger** – 800 rows of raw trade data (Manager, Sector, P&L, etc.)
- **Alpha Report** – Calculated metrics: Sharpe, Sortino, Calmar ratios, VaR
- **Pivot Analysis** – Multi-dimensional pivot tables + charts
- **Executive Dashboard** – KPI summary for leadership view

## 🛠️ Skills Demonstrated
- Excel Pivot Tables & Slicers
- VLOOKUP
- Conditional Formatting
- KPI Dashboard Design
- Financial Metrics (Sharpe Ratio, VaR, Max Drawdown)
- Data Storytelling
